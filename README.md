# BeautyShop E-Commerce Project

This project was created during the DecodeLabs Internship.

It is a simple e-commerce shopping cart website.  
The frontend shows beauty products and the backend provides an API to get and add products.

The backend is connected to a PostgreSQL database.

Technologies used:
HTML
CSS
JavaScript
Node.js
Express.js
PostgreSQL


How to run the project:

1. Start backend

cd project-2-backend-api
node server.js

2. Open browser and check API

http://localhost:5000/products

3. Open the frontend

Open index.html from project-1-responsive-ui folder.
